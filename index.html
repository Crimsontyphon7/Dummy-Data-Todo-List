First Part of the Exercise

Three buttons that call functions to:
1) Perform the fetch
2) Log the results to the console
3) Populate an ordered list with the fetch results

NOTE: Fetch and Log before testing the Second and Third parts of the exercise.

Second Part of the Exercise

1) Form to input a number between 1 and 10
2) Button to call function to filter ToDos for that UserID and populate ordered list with the results
3) Button to clear the list from the HTML page.

Third Part of the Exercise

1) Button to call function to filter completed ToDos
2) Button to call function to filter not completed ToDos
